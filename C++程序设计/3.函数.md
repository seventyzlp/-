## 函数定义与使用

函数名+参数

```cpp
void function( int xxx,xy ){
print("111");
}
//类型表示符 如果没有返回值那么就为void，无return；若有return 那么需要和返回值的数据类型相同
//function后有形式参数表
```

- 函数调用的形式 函数名（实参列表）
    - max(a,b,c)
- 如果使用库函数，还要在文件头include
- 或者可以写在头文件里面，也需要include
- 使用自定义函数，如果函数的位置在调用它的函数的后面，需要在调用它的函数**声明（就是在前面先定义一下函数）**
    - 声明不需要函数的完整结构，只是告诉系统需要在内存上分配空间，与定义不同，定义是直接在内存里写东西
    - 函数声明是对返回值的说明，通知系统
    - 不包含函数题
    - 调用几次函数久在各个函数中做声明
    - 以分号结束
- 函数定义
    - 定义只有一次
    - 花括号不加分号
    - 包含所有内容
    - 可以有默认值
        - 有参数有默认值，有参数无，无默认值参数必须在最左侧
        - 从左向右开始引用参数
        - 如果需要先声明函数，那么在声明时说明默认值
        - 推荐定义在main之前

```cpp
#include <iostream>
using namespace std;

double xplusplus(double _x, int _n); //这个就是声明

int main()
{
    int x, y;
    cout << "输入底数:";
    cin >> x;
    cout << "输入指数:";
    cin >> y;
   cout << "答案是" << xplusplus(x, y) << endl; 
    return 0;
}

double xplusplus(double _x = 1, int _n = 2)//如果想要返回数组那就需要为array，默认值
{//如果没有参数那就会按照默认值，如果有参数就会覆盖
    int _result;
    _result = _x;
    for (int i = 1; i < _n; i++)
    {
        _result *= _x;
    }
    //cout << "答案是" << _result << endl;
    return _result;
}
```

### 函数的调用

    - 在一个函数执行中，调用别的函数
- 直接调用
- 递归调用
    - 直接或者间接调用自身，但是实际参数值是不一样的

## 函数的参数

函数名+函数类型

- 实际参数：在调用时传递给函数的参数
    - 常量、变量、表达式
    - 必须有确定的值
    - 预先赋值
    - 不能在函数内部使用
- 形式参数：不存在，是虚拟变量
    - 定义函数的时候用的
    - 接受传入函数的参数
    - 一一对应
    - 函数可以没参数，但是一定有返回值
    - 只能在函数内部使用
- 形参和实参是不同的内存地址，复制一份，形参会在运行结束后被释放

    - 值传递
        - 将实际参数的值复制到形式参数中
        - 任何对形式参数的修改不会影响实际参数的值

    - 指针传递
    - 引用传递
        - 实际参数和形式参数的内存地址相同

## 内联函数

- 在线函数
- 相当于把函数体的内容直接复制到main函数中进行操作“镶嵌”
- 节省参数传递、控制转移的时间，但是占用内存大
- 声明时要使用inline
- 内联函数一般在头文件中定义(.h)文件
- 不实用于复杂计算，递归
- 定义必须在第一次调用之前
- 不能有switch和循环，可以if（不能嵌套）

## 函数的重载

- 相近功能的函数可以名字一样，但是需要形参类型或者数量不同
- 但是函数体还是要自己写，只是方便归类使用
- 返回值、形式参数名字相同也没关系

```cpp
swap(int a,int b){
}
swap(double a,double b){
}
//通过参数决定调用的函数
```
