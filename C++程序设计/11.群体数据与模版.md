# 函数模版

如果重载的函数，解决问题的逻辑是一致的，函数体语句相同，只是处理的数据类型不同，那么写多个是浪费，并且因为代码冗余造成不一致性->使用模版

- 创建一个通用功能的函数
- 支持多种不同形参
- 简化重载函数的函数体设计

## 求绝对值函数的模版

```cpp
template <typename T>
T abs(T x){
	return x<0 ? -x:x;
}
```
自动将模版转换为想要调用的明确函数

## 函数模版定义语法

- 语法形式
	- template<模版参数表>
	- 函数定义
- 模版参数表的内容
	- 类型参数:：class、typename
	- 常量参数：类型说明符、标识符
	- 模版参数：template<参数表> class 标识符
